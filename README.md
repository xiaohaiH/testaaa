## 运行

> -   安装依赖(`pnpm i`)
> -   启动服务
>     -   预览指定版本(`npm start`), 默认 `vue3`
>     -   同时预览多个版本(注: 需将所有模块都打包(`element-plus` & `element-el` & `example`))
>         -   开启预览(`pnpm preview`)
>         -   开启`example`文件监听自动重新打包(`pnpm lib`)
